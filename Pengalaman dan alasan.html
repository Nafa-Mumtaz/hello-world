Pengalaman di ITB dan alasan masuk ARC



Pengalaman saya di ITB sebelum masuk ARC, saya sudah mendaftar di 2 unit lain, yaitu Keluarga Mahasiswa Islam (GAMAIS) dan Unit Kebudayaan Tionghoa (UKT). Selain itu, saya juga mengikuti kepanitiaan Aku Masuk ITB (AMI) di bagian Ekstrakampus media partner dan ITB Goes To School (IGTS).

Di GAMAIS, saya banyak mempelajari tentang agama saya yaitu Islam. Saya benar-benar merasakan manfaat dari keikutsertaan saya di GAMAIS, yaitu pemahaman saya mengenai agama saya meningkat. Selain itu, rasa persaudaraan di GAMAIS juga tinggi.

Di UKT, saya banyak belajar mengenai kebudayaan Tionghoa. Saya memilih untuk mendalami Kung Fu di UKT, dan saya merasakan dampak baik, yaitu kesehatan saya membaik. Selain itu, saya mendapatkan kemampuan bela diri dasar yang sangat berguna bagi saya.

Karena banyaknya manfaat yang saya dapatkan dari mengikuti unit di ITB, saya percaya bahwa ARC juga akan berguna bagi saya ke depannya. Saya pertama kali mengetahui tentang ARC dari mentor saya di OSKM, lalu saya mendapat banyak informasi darinya mengenai ARC. Saya menceritakan kepadanya tentang keinginan saya untuk memasuki jurusan informatika dan menanyakan kepadanya mengenai saran unit yang cocok, lalu ia menyarankan ARC. Setelah itu saya mencari informasi tambahan mengenai ARC, dan merasa bahwa ARC akan sangat membantu saya dalam mendalami dunia komputing, terutama coding, sehingga saya akhirnya memutuskan mendaftar ARC saat OHU.
